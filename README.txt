Document Scanner with Optical Character Recognition .
OCR is implemented using tesseract .

Downloads needed :
1. tesseract (check for the Github Account for tesseract)
2. pip install pytesseract
3.pip install pillow

Instructions to run :
1.Once downloaded the tesseract , copy the two python files into the Tesseract-OCR folder in the Program Files.
2.Now open the terminal and run the file scanV3.py .


Shortcomings :
1.The document scanner is not very exact and produce wrong outputs if the borders are not well defined . Prefer drawing the bold boundaries.
2.tesseract though produce very accurate results , but likely to produce errors in case the background are noisy .
